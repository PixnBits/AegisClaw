# Agents



For start and stop controls:
* use `$ sudo ./bin/aegis start &> aegis.log` (or `make start`) to start the daemon (no password needed for the sudo command)
  * `$ sudo ./scripts/build-microvms-docker.sh` also does not need a password for the sudo command
* use `$ ./bin/aegis stop` (or `make stop`) to stop the daemon (no sudo needed)
