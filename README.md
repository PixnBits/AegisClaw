# AegisClaw v2

**Status:** Phase 0 - Foundations & Testing Infrastructure

AegisClaw is a secure, sandboxed AI agent runtime built for safety and reliability.

## Quick Start

### Building

```bash
make build
```

Or build individual components:

```bash
make build-binaries
```

### Running

```bash
# Start the daemon
./aegis start

# Check status
./aegis status

# Run health checks
./aegis doctor
```
